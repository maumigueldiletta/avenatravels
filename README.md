# avenatravels.com

Site institucional da **Avena** — sua agência de viagem com IA.

- `index.html` — landing principal
- `privacidade.html` — Política de Privacidade (link da App Store)
- `termos.html` — Termos de Uso (EULA)
- `suporte.html` — Suporte (link da App Store)
- `assets/` — robô, logo, ícone, screenshots e `site.css`

Estático puro. Deploy via Vercel (domínio: avenatravels.com).
